<objective>
Integrate the Next.js frontend with the deployed `UBTC` and `CollateralManager` smart contracts. This includes adding wallet connection functionality, displaying token balances, and creating a user interface for minting new `UBTC` tokens.

Purpose: To create a functional user-facing dashboard for the stablecoin system.
Output: A web interface where users can connect their wallet, view their `UBTC` balance, and mint new tokens by depositing collateral.
</objective>

<task type="auto">
  <name>Task 3: Implement Minting Form and Transaction</name>
  <files>
    - "src/components/MintForm.tsx"
    - "src/app/page.tsx"
  </files>
  <action>
  1. Create `src/components/MintForm.tsx`.
  2. The component will contain a number input for the amount of collateral (tRBTC) to deposit and a "Mint UBTC" button.
  3. Use Wagmi's `useWriteContract` hook to prepare the transaction. The hook should be configured to call the `deposit` function on the `CollateralManager` contract.
  4. The `value` for the transaction will be the amount entered by the user, converted to a BigInt (wei).
  5. When the user clicks the button, call the function returned by `useWriteContract` to trigger the transaction in their wallet.
  6. Add the `MintForm` to `src/app/page.tsx`. Wagmi's provider will automatically handle re-fetching the balance after the transaction succeeds.
  </action>
  <verify>
    <automated>A user can enter an amount, click "Mint UBTC", and be prompted by their wallet to confirm a transaction. After confirmation, the `UBTC` balance displayed in `WalletInfo` should update to the new amount.</automated>
  </verify>
  <done>
    A user can successfully mint new `UBTC` tokens through the frontend dashboard.
  </done>
</task>
